## Navigation Patterns

### Pattern 1: New to the project
```
1. Read README.md (project overview)
2. Read docs/vision.md (understand goals)
3. Read docs/architecture.md (see how it works)
```
**Tokens:** ~5,850 vs 20,000 (70% savings)

### Pattern 2: Want to implement an agent
```
1. Read docs/agents/overview.md (see all agents)
2. Read specific agent doc (e.g., trader.md)
3. Read docs/design-principles.md (coding standards)
```
**Tokens:** ~9,600 vs 20,000 (52% savings)

### Pattern 3: Quick reference
```
1. Read NORTH_STAR.md (links to everything)
```
**Tokens:** ~1,800 vs 20,000 (91% savings)

### Pattern 4: Check implementation progress
```
1. Read IMPLEMENTATION_STATUS.md
2. Read docs/implementation-plan.md
```
**Tokens:** ~6,000 vs 20,000 (70% savings)
