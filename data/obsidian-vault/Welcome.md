# Welcome to Life AI Knowledge Base

This is your test Obsidian vault for the Life AI project.

## Getting Started

The Note-taker agent will automatically:
- Create notes from your input
- Add frontmatter (tags, dates, titles)
- Generate `[[wiki-links]]` to related notes
- Organize notes into categories
- Build a knowledge graph

## Categories

- `trading/` - Trading analysis and journals
- `learning/` - Learning notes and curricula
- `japanese/` - Language learning
- `projects/` - Project documentation
- `meetings/` - Meeting notes
- `ideas/` - Quick thoughts
- `daily-notes/` - Daily journal

Start capturing knowledge! 🚀
